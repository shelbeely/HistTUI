# HistTUI Screenshots

This directory contains screenshots of HistTUI features captured as SVG animations.

## Screenshots

1. **01-setup-wizard.svg** - First-time setup wizard with LLM configuration
2. **02-activity-dashboard.svg** - Activity Dashboard (default screen)
3. **03-timeline.svg** - Commit Timeline view with navigation
4. **04-commit-detail.svg** - Commit Detail screen showing diffs
5. **05-branches.svg** - Branches & Tags view
6. **06-file-tree.svg** - File Tree Explorer
7. **07-repo-manager.svg** - Multi-Repository Manager (Press '5')
8. **08-code-planner.svg** - Code Planner for agent-driven development (Press '6')
9. **09-search.svg** - Fuzzy Search interface (Press '/')

## Viewing Screenshots

These are SVG animations that can be viewed in:
- Modern web browsers
- GitHub README files
- Documentation sites

## Capture Details

- Terminal size: 120x35
- Captured using: asciinema + termtosvg
- Theme: Material Design 3 (Primary: #6750A4)
